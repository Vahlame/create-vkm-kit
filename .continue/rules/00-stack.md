## Stack (repo)

- **Languages:** Markdown (vault + docs), TypeScript (tooling), Go 1.22+ (`obsidian-memoryd`), Python 3.11+ (optional RAG).
- **Runtimes:** Node 20+, Bun or `npx tsx` for maintainer scripts; `uv` for `basic-memory`.
- **Primary MCP:** `uvx basic-memory mcp` with `BASIC_MEMORY_HOME=<vault>` (**stdio** in `mcp.json`; recommended default).
- **Optional transport:** Streamable HTTP URL for `basic-memory` (e.g. `http://127.0.0.1:8765/mcp`) requires a **separate always-on listener** you run (terminal or Task Scheduler action you define; see `docs/en/sync.md`); see `config/mcp/basic-memory-streamable-http.json`. Default port **8765** avoids collisions with common dev servers on **8000** / **8080** / **3000** (see ADR-0016).
- **Hybrid MCP (recommended vault surface):** `node packages/obsidian-memory-mcp/src/hybrid-mcp.mjs` — twenty-two tools: `vault_hybrid_search` (BM25 + semantic via RRF; returns the matching **section** — passage-first; all-opt-in/default-off knobs: `graph: true` fuses `[[wikilink]]`-adjacent notes — ADR-0019, `graphTyped: true` verb-weights typed relations — ADR-0027, `recency: true` favors recently-modified notes — ADR-0021, `importance: true` favors hub notes by in-degree — ADR-0027, `mmr: true` diversifies near-duplicates — ADR-0028, `passageWindow: N` widens the returned passage, `rerank: true` adds a cross-encoder precision pass — ADR-0026, needs the `[rerank]` extra; compact wire format by default, `explain: true` restores ranking diagnostics and default `limit` is 10 — ADR-0034; CLI/env levers `--pin-failures` / `--usage` boost recorded `[failure]`/`[gotcha]` lessons and demonstrably-used notes — ADR-0038), `vault_fts_search` / `vault_fts_index`, `vault_complete` (prefix autocomplete over titles / filenames / `#tags`), the structured knowledge graph `vault_relations` / `vault_observations` / `vault_kg_suggest` (typed relations + categorized observations — ADR-0023), `vault_memory_report` (indices + hygiene/compaction candidates + `cold_notes`, `reflect: true` adds consolidation proposals, read-only — ADR-0024/0038), vault-locked `vault_read_file` (returns a content `etag`) / `vault_write_file` / `vault_edit_file` (opt-in `ifMatch` precondition + advisory write lock — ADR-0037; every write/edit/append also runs a **write-time hygiene lint** — non-canonical observation categories, unresolved `[[wikilinks]]`, orphan-at-birth notes — warning-only, scoped to the text being written) / `vault_delete_file` (soft delete to `.trash/` by default; `permanent: true` unlinks; refuses core protocol notes; reports notes still `[[linking]]`) / `vault_move_file` (rename/move; refuses overwrite without `overwrite: true`; reports stale `[[wikilink]]` refs; restore-from-trash is a move back) / `vault_append_file` (CRLF-aware append — the SESSION_LOG one-liner path, no anchor round-trip) / `vault_frontmatter_set` (top-level scalar YAML keys without text-matching — `status`/`last_verified` promotions) / `vault_backlinks` (who `[[links]]` here, read-only, pre-delete/move impact check) / `vault_git_history` (commits + old versions of a note via the sync repo, read-only — recovery even after `permanent: true`) / `vault_list_directory`, `vault_rotate_log` (archive old `SESSION_LOG` entries — `##` sections or top-level bullets — keeping the newest N; bridges Python `json-rotate-log`), `vault_audit` (vault health: bloat, broken links, sync conflicts, git state, opt-in `memory-schema.json` checks), `memory_extract_candidates`, `assemble_context` (one budgeted call: decisions + stack + passages for a project). Bridges Python `obsidian-memory-rag` (optional `[vec]` extra + `OBSIDIAN_MEMORY_SQLITE_VEC=1` accelerates the cosine scan in-file via sqlite-vec, ranking-identical — ADR-0025); see `config/mcp/obsidian-memory-hybrid.json`. Wire it in Claude Code with `create-vkm-kit --ide claude` (runs `claude mcp add`) or Codex CLI with `--ide codex` (runs `codex mcp add`); `create-vkm-kit --full` wires Codex + Claude with hybrid+semantic+index+rules in one shot (ADR-0022).
- **Optional MCP (Obsidian live I/O):** `cyanheads/obsidian-mcp-server` (Streamable HTTP `/mcp`) with path allowlists.
- **Bridge (legacy clients):** `mcp-remote` pinned **>= 0.1.16** (see `docs/security/mcp-remote-rce.md`).
